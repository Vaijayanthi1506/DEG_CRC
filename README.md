# DEG_CRC
The code available in this repository are for the project title 'Comparative analysis of the gene signatures between Bacteroides fragilis and Fusobacterium nucleatum over Colorectal adenocarcinoma"
The RNA-Seq and microarray dataset used in the manuscript are available in the Gene Expression Omnibus database under accession numbers: GSE102573 (https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE102573) and GSE113218 (https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE113218). 

The entire project was performed using R language, R studio version 2024.12.1+563.
Packages used in the manuscript includes, limma, edgeR, oligo,limma, pd.hta.2.0, clusterProfiler, enrichplot, ggplot2, DOSE,org.Hs.eg.db, ReactomePA.
